Primary call-to-action — uppercase, widely-tracked label; brass gradient fill by default. Use for "Add to cart", "Checkout", "Shop the range".

```jsx
<Button variant="primary" size="lg" onClick={checkout}>Proceed to checkout</Button>
<Button variant="outline">View details</Button>
<Button variant="ember">Checkout</Button>
```

Variants: `primary` (brass gradient, default), `ember` (oxblood secondary), `outline` (gold hairline), `ghost` (text-only). Sizes `sm | md | lg`. `fullWidth`, `disabled`, `iconLeft`/`iconRight` supported.
